# Drone Map Viewer

A React + Vite + TypeScript viewer for 3D maps exported from Agisoft Metashape (drone photogrammetry).

## Quick start

```bash
cd drone-map-viewer
npm install
npm run dev
```

Then open http://localhost:5173, drag-and-drop or browse for a `.glb` or `.gltf` file exported from Metashape.

## Stack

- React 18 + TypeScript (strict mode)
- Vite 5
- Three.js via @react-three/fiber + @react-three/drei
